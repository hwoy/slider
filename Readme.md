### Slider is a cross platform slider game.



- How to build

```sh
$ make
```
- For 3x3 ABC

```sh
$ make -f Makefile.3
```

- For 4x4 ABC

```sh
$ make -f Makefile.4
```

- For 5x5 ABC

```sh
$ make -f Makefile.5
```

- For All

```sh
$ ls Make* | xargs -I{} make -f {}
```
